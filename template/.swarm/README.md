# .swarm

This folder is installed by AI Swarm Cockpit.

Useful commands:

```bash
.swarm/bin/swarm-start
.swarm/bin/swarm-stop
.swarm/bin/swarm-status
.swarm/bin/watch-chat
.swarm/bin/swarm-boot
```

Agent communication:

```bash
say claude "message"
say codex "message"
say deepseek "message"
say agy "message"
broadcast "message"
```
